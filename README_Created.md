# Weather Dashboard

  [![License](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

  ## Description
  
  Allows users to submit a location and receive the weather currently, as well as a 5 day forecast. Building this allowed practice in JS, CSS, and HTML, as well as practice in the use of APIs.
